IMO 2024/2
Determine all pairs $(a,b)$ of positive integers for which there exist positive integers $g$ and $N$ such that \[ \gcd(a^n + b, b ^n + a) = g \] holds for all integers $n \ge N$.
IMO 2024/2 (N4)

IMO Shortlist 2022 A6
Let $\mathbb{R}$ be the set of real numbers. We denote by $\mathcal{F}$ the set of all functions $f: \mathbb{R} \to \mathbb{R}$ such that \[ f(x+f(y)) = f(x) + f(y) \] for every $x, y\in \mathbb{R}$. Find all rational numbers $q$ such that for every function $f \in \mathcal{F}$, there exists some $z \in \mathbb{R}$ satisfying $f(z) = qz$.
